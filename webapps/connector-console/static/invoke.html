Invoke Resource

選択したユーザの秘密鍵で署名し、Connector の /invoke_resource を呼び出します。

リソース呼び出し時に付与するクエリパラメータを指定します。"room_id=roomA" または "?room_id=roomA" のどちらの形式でも入力できます。複数指定する場合は "room_id=roomA&light_id=a302-1" のように "&" で連結してください。入力値は /invoke_resource のクエリ文字列として Provider 側 Connector に中継されます。

任意のHTTPヘッダーを1行1件で指定できます。形式は "Header-Name: value" です。署名用の X-Resource-Id / X-User-Id / X-Expire-Time / X-Signature と Authorization は専用欄で扱うため、ここでは指定しません。

Provider 側リソースが HTTP Authorization ヘッダーを要求する場合だけ指定します。未指定の場合は従来通り Authorization ヘッダーを付与しません。
